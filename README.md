# 💰 Budget Tracker

A simple personal finance app built with Spring Boot that helps you follow the **50/30/20 budgeting rule**. To tracker your income flows and manage your expenses

## ✨ Features

- Track income and expenses
- Automatic 50/30/20 breakdown (Needs / Wants / Savings)
- Smart warnings when you overspend
- Pie chart visualization
- Edit & Delete transactions
- Monthly navigation
- Export to CSV

🛠 Tech Stack

Spring Boot 3 + Java 21
Thymeleaf + Chart.js
H2 Database (embedded)

How It Works

-Add your income
-Log expenses under Needs, Wants, or Savings
-Get automatic insights + warnings
